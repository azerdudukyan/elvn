elvn

elvn - time management for nerds

download elvn.jar view on github

track your time right now

after downloading jar you can run it with command.

$ java -jar elvn.jar

later you can save this command into bash/cmd file and place to your environment to run it with simple "elvn" command. after running elvn wil show you a list of planned tasks and interactive command prompt. learn more about elvn ideology here

Lists

elvn provides user with opportunity to manage 7 projects organized in lists of tasks and 3 system lists.

system lists are :
  • today - planned tasks
  • all - all tasks
  • done - completed tasks
project lists are indentified by color name and label if set.
  • red - not assigned
  • orange - not assigned
  • yellow - not assigned
  • green - personal
  • blue - work
  • pink - not assigned
  • violet - not assigned
by default only blue (work) and green (personal) lists are enabled. to enable list you need to assign label to list.
